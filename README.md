# Stack-Based Expression Evaluator

## Overview
This project implements a **stack-based mathematical expression evaluator** in Python. It reads expressions from an input file (`input.txt`), evaluates them using stacks, and writes the results to an output file (`output.txt`).

## Files
- `stack_evaluator.py` — Main Python program.
- `input.txt` — Input expressions.
- `output.txt` — Generated output file.
- `README.md` — Project documentation.

## How to Run
1. Ensure Python 3 is installed on your system.
2. Place all files in the same directory.
3. Run the following command in your terminal:
   ```bash
   python stack_evaluator.py
   ```
4. The evaluated results will appear in `output.txt`.

## Example
### input.txt
```
5+8
-----
(8/4)+7*2
-----
10-(2+3)*4
-----
```

### output.txt
```
13
-----
17
-----
-10
-----
```
